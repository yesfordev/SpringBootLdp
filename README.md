### ✨Overview

✔️부동산 데이터 수집 API Server

회사에사 시작하는 프로젝트의 일환으로, 가공할 데이터를 수집하는 작업이 필요했습니다. 공공 데이터 API를 이용하여 약 50만 건의 전국의 부동산 거래 정보, 창고정보 등을 수집한 후 저장하는 API를 구현하였습니다.
